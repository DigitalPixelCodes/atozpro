| **Runway Gen-3 Alpha** | Realistic human scenes, smooth motion | Best quality for airport, people, lifestyle shots |

| **Kling AI** | Longer clips, realistic movement | Good for full 4-5s continuous shots |

| **ElevenLabs** | Voiceover generation | Use warm, professional, slight British-English accent |
| **Suno AI** | Background music | Generate custom royalty-free travel track |

## Suggested Workflow

1. **Generate images** using prompts above (Midjourney / DALL-E) for static references
2. **Generate video clips** using Runway or Kling with the video prompts (export each scene separately)
3. **Record VO** with ElevenLabs or real voice — use the exact VO lines from each scene
4. **Assemble in CapCut**: Import clips → add VO track → lay text overlays → add background music + SFX → color grade warm
5. **Export** in 1080×1920 for Reels/Shorts AND 1920×1080 for YouTube
6. **A/B test** two versions: one with real human faces, one with animated/illustrated style
